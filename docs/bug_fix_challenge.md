## ریشه‌ی مشکل
زیرنویس‌ها از ابزارهای ترجمه‌ی مبتنی بر مرورگر استخراج شده بودند.
در این خروجی:
- هر کلمه داخل یک `<span>` جداگانه قرار داشت
- جملات کامل داخل attribute به نام `data-sentence` ذخیره شده بودند
- کاراکتر `+` برای عملکرد ذخیره‌ی جمله اضافه شده بود

این ساختار باعث تخریب خوانایی و پردازش‌پذیری زیرنویس می‌شد.

---

## استراتژی رفع باگ
فرآیند پاک‌سازی شامل مراحل زیر است:
1. حذف کامل تمام تگ‌های HTML
2. استخراج متن خام بدون وابستگی به attributeها
3. حذف صریح تمام کاراکترهای `+`
4. نرمال‌سازی فاصله‌ها و line breakها

---

## نتیجه
پس از این مرحله:
- متن زیرنویس کاملاً قابل خواندن شد
- آماده ورود به مرحله‌ی همزمان‌سازی گردید
- هیچ وابستگی به ساختار HTML باقی نماند
